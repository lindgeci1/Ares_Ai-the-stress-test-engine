# ARES AI — Tech Stack

## Frontend
- **React 18** with **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS 3.4** (styling)
- **React Router v6** (routing + protected routes)
- **Axios** (HTTP client with interceptors for JWT refresh)
- **Stripe.js + React Stripe** (client-side payment UI)
- **Lucide React** (icons)

## Backend
- **Go (Golang)** with **Fiber v2** (HTTP framework)
- **GORM** (ORM + auto-migration)
- **golang-jwt/v5** (JWT access + refresh tokens)
- **bcrypt** (password hashing)
- **Swagger** via swaggo/fiber-swagger (API docs)
- **godotenv** (environment config)
- **Ollama API** (debate generation and resilience analysis)
- **Google Cloud Text-to-Speech** (dual-voice debate synthesis)

## Database
- **PostgreSQL** (via Supabase)
- 10 tables: users, roles, user_roles, user_usages, refresh_tokens, documents, audit_reports, audio_debates, offers, payments

## External Services
- **Stripe** (stripe-go/v82) — payment intents, checkout
- **Cloudinary** (cloudinary-go/v2) — document file upload/storage

## AI Services
- **Ollama API** — debate generation, vulnerability analysis, resilience scoring, heatmap, fallacies, fortification plan
- **Google Cloud Text-to-Speech** — dual-voice audio debate (Auditor + Optimist)
