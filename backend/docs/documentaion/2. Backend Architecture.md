# ARES AI — Backend Layered Architecture

## Project Root

```
backend/
├── cmd/api/main.go            # Entrypoint — DB, Stripe, Fiber init, DI wiring, seed, migrate
├── docs/                      # Auto-generated Swagger (docs.go, swagger.json, swagger.yaml)
└── internal/
    ├── middleware/             # JWT auth guards
    │   └── auth_middleware.go  # RequireAuth(), RequireRole("Admin"|"User")
    ├── models/                 # GORM structs (DB schema + request/response DTOs)
    │   └── user.go            # User (includes is_temp/access_key/expires_at), Role, UserRole, UserUsage (includes rounds_per_audit), RefreshToken, RegisterRequest, LoginRequest (email+password or email+access_key), LoginResponse, UpdateUserRequest
    │   ├── document.go        # Document, AuditReport, AudioDebate, UpdateDocumentRequest
    │   ├── payment.go         # Payment
    │   └── offer.go           # Offer
    ├── repository/            # Data access layer (GORM queries)
    │   ├── user_repository.go
    │   ├── document_repository.go
    │   ├── payment_repository.go
    │   └── offer_repository.go
    ├── service/               # Business logic layer
    │   ├── user_service.go
    │   ├── document_service.go
    │   ├── payment_service.go
    │   ├── offer_service.go
    │   └── cloudinary_service.go  # External: file upload/delete
    ├── handlers/              # HTTP layer (Fiber handlers)
    │   ├── admin_handler.go   # Admin-only aggregated stats handler
    │   ├── user_handler.go
    │   ├── document_handler.go
    │   ├── payment_handler.go
    │   ├── offer_handler.go
    │   └── health_handler.go
    └── routes/                # Route registration + middleware binding
        ├── auth_routes.go
        ├── user_routes.go
        ├── document_routes.go
        ├── payment_routes.go
        ├── offer_routes.go
        └── health_routes.go
```

## Call Chain (Who Calls Who)

```
main.go
  │
  ├─ GORM AutoMigrate (models)
  ├─ seedRoles(), seedOffers()
  │
  ├─ NewXxxRepository(db)        ──► Repository layer
  ├─ NewXxxService(repo)         ──► Service layer
  ├─ NewXxxHandler(service)      ──► Handler layer
  │
  └─ SetupXxxRoutes(api, handler) ──► Route layer
```

**Per-request flow:**

```
HTTP Request
  │
  └─► Route (defines path + middleware)
        │
        └─► Middleware (RequireAuth / RequireRole)
              │
              └─► Handler (parse request, extract JWT claims, call service, return JSON)
                    │
                    └─► Service (validate, business logic, orchestrate)
                          │
                          ├─► Repository (GORM DB queries)
                          └─► External Service (Cloudinary, Stripe SDK, Ollama API, Google Cloud TTS)
```

## Domain Wiring (main.go)

| Domain    | Repository              | Service                | Handler              | Routes               |
|-----------|-------------------------|------------------------|----------------------|-----------------------|
| Auth/User | UserRepository          | UserService            | UserHandler          | auth_routes, user_routes |
| Documents | DocumentRepository      | DocumentService        | DocumentHandler      | document_routes      |
| Offers    | OfferRepository         | OfferService           | OfferHandler         | offer_routes      |
| Payments  | PaymentRepository       | PaymentService         | PaymentHandler       | payment_routes      |
| Admin     | DocumentRepository / DB  | N/A                    | Admin handler fn     | document_routes / offer_routes / payment_routes |

**PaymentService** is the only service that injects multiple repositories: `PaymentRepository`, `OfferRepository`, `UserRepository`.

**DocumentService** optionally injects `CloudinaryService` for file uploads.

## Middleware Guards

| Middleware        | Validates                        | Used On                                     |
|-------------------|----------------------------------|---------------------------------------------|
| `RequireAuth()`   | Valid JWT (Admin or User)        | `/auth/me`, `/payments/*`                   |
| `RequireRole("Admin")` | Valid JWT + role == Admin (bypass for Admin on all) | `/users/*`, `/admin/*`             |
| `RequireRole("User")`  | Valid JWT + role == User (Admin also passes)       | `/documents/*`                     |
| *(none)*          | Public                           | `/auth/login`, `/auth/register`, `/auth/logout`, `/auth/refresh`, `/offers`, `/health` |
